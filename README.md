# Gulf Uni Guide AI (GCC University Matcher)

AI assistant for Gulf high-school students to discover suitable universities and admission requirements across GCC (e.g., Saudi student wants to study in Qatar).

## Features (MVP)
- Student profile intake (grades, interests, budget, language, preferred country/city)
- AI-based matching + explanation (why each university/program fits)
- Requirements summary (tests, language scores, documents)
- Output as shortlist + checklist
